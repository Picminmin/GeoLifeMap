# GeoLifeMap

世界各国の **出生時平均寿命 (Life Expectancy at Birth)** の変化を
地図とランキングを組み合わせて可視化するプロジェクトです。

WHOが公開しているデータを利用して
**2000年〜2021年の平均寿命の推移**をアニメーションとして表示します。

# 可視化の内容

このアニメーションでは次の2つを同時に表示しています。

### 🌍 世界地図

- 国ごとの平均寿命を色で表示
- 色が濃いほど平均寿命が高い

### 📊 ランキング

- 平均寿命 **Top40** を表示
- 大陸ごとに色分け
